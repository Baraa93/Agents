# Project Constitution

## Product Rules
- Mobile-first
- Clean modern UI
- Premium feel, not generic
- Fast loading
- Clear CTA on every main page

## Code Rules
- Use reusable components
- Keep files organized
- Avoid messy logic
- Separate UI from backend logic
- Keep naming clean and predictable

## Design Rules
- Strong spacing
- No boring plain sections
- Good hierarchy
- Modern cards, layout, and motion
- Avoid excessive animation

## Business Rules
- Scalable structure
- Easy to maintain
- Ready for integrations
- SEO-friendly from the start
